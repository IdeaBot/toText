# toText
IdeaBot add-ons package for converting things to text.
Currently works with images, but in the future this may work with speech as well.

## Overview ##

### Commands ###
This outlines the functionality of each command.
Use `@Idea help <command name>` to get usage instructions.

* img2txt
> Pull the text out of an image.

* ocr_plugin
> Plugin back-end for img2text.
> This uses optical character recognition (OCR) to convert an image to text.
